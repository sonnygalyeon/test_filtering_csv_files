# README

## Запуск bash-скрипта

Сначала сделайте скрипт исполняемым:

```bash
chmod +x main.sh
```

После этого запустите его:

```bash
./main.sh
```

Альтернативный вариант запуска без выдачи прав:

```bash
bash main.sh
```

Скрипт `main.sh` активирует виртуальное окружение `venv` и запускает `filter_script.py` с файлами `math.csv`, `physics.csv`, `programming.csv` и отчетом `median-coffee`.

---

## Что делает `filter_script.py`

Скрипт читает несколько CSV-файлов с данными по студентам за период сессии и формирует единый отчет.

В текущем варианте используется отчет:

- `median-coffee` — медианная сумма трат на кофе по каждому студенту по всем переданным файлам сразу.

То есть данные не считаются отдельно по каждому CSV, а объединяются в один общий набор.

---

## Логика работы

1. Скрипт получает список CSV-файлов через параметр `--files`.
2. Получает название отчета через параметр `--report`.
3. Читает строки из всех переданных файлов.
4. Группирует данные по имени студента.
5. Для каждого студента собирает все значения из колонки `coffee_spent`.
6. Считает медиану этих значений.
7. Сортирует результат по убыванию медианы.
8. Выводит готовую таблицу в консоль.

---

## Пример команды, которую выполняет скрипт

```bash
python3 filter_script.py --files math.csv physics.csv programming.csv --report median-coffee
```

---

## Формат входных данных

Ожидается CSV с колонками:

```csv
student,date,coffee_spent,sleep_hours,study_hours,mood,exam
```

Для отчета `median-coffee` используется колонка `coffee_spent`.

---

## Что такое медиана

Медиана — это центральное значение в отсортированном списке.

Пример:

- значения: `200, 300, 500`
- медиана: `300`

Если значений четное количество, берется среднее двух центральных.

Пример:

- значения: `200, 300, 500, 700`
- медиана: `400`

## Пример результата

На выходе скрипт печатает таблицу вида:

```text
+------------------+-----------------+
| student          |   median_coffee |
+==================+=================+
| Иван Кузнецов    |             700 |
| Дмитрий Морозов  |             610 |
| Михаил Павлов    |             590 |
+------------------+-----------------+
```
